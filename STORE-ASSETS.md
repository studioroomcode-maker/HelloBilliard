# 스토어 등록용 이미지·문구 자료 (Play + App Store 공용)

스크린샷 찍는 순서, 캡션 문구, 피처 그래픽 시안까지 이 문서만 보고 만들면 된다.

---

## A. 스크린샷 — 무엇을 몇 장 찍나

**추천 5장 (순서 = 스토어에 보이는 순서. 첫 장이 전환율의 대부분을 결정한다)**

| # | 화면 | 만드는 법 | 캡션(이미지 위에 얹을 문구) |
|---|---|---|---|
| 1 | **사진 인식 결과** | 4구 탭 → [사진 인식] → 당구대 사진 → 인식 완료(사진 위 경로 오버레이) | `사진 한 장이면 끝` / `공 배치를 자동으로 읽어요` |
| 2 | **4구 경로 + 성공률 카드** | 4구 탭 → [경로 계산] → 카드 여러 개 뜬 상태 | `성공률까지 계산해 주는 AI 코치` |
| 3 | **3구 정석 코스** | 3구 탭 → [경로 계산] → `대회전`/`와리가리` 배지 뜬 카드 | `대회전·와리가리… 정석 코스를 알려줘요` |
| 4 | **3구 시스템 계산기** | 3구 탭 → [파이브앤하프] → 레일 숫자 표시된 화면 | `파이브앤하프·플러스·볼 시스템` |
| 5 | **실력별 난이도 / 내 기록** | 개인 설정 펼친 화면 또는 [기록] 모달 | `내 실력에 맞춘 추천` |

### 찍는 방법 (권장: 실기기 대신 브라우저)
1. 크롬에서 앱 열기 → `F12` → **기기 툴바(Ctrl+Shift+M)**
2. 해상도 지정:
   - **Play**: 1080×1920 (9:16)
   - **App Store 6.7"**: **1290×2796** (필수)
   - **App Store 6.5"**: **1242×2688** (필수)
3. 각 화면 세팅 후 `Ctrl+Shift+P` → `Capture screenshot`
4. 캡션은 Figma/Canva에서 위쪽 15~20% 영역에 얹기 (배경 `#100e0a`, 글자 `#f4f1e8`, 강조 `#e8b53e`)

> ⚠ 스크린샷에 **Pro 구매 버튼이 보이면 안 된다**(스토어 정책). 찍기 전에 콘솔에서
> `__hbSetStore('play')` 실행 → 구매/후원 버튼이 사라진 상태로 촬영.
> 되돌리려면 `__hbSetStore(null)`.

## B. 피처 그래픽 (Play 전용, 1024×500)

**레이아웃 시안**
```
┌──────────────────────────────────────────────────┐
│  [배경: #100e0a → #1a1510 대각 그라데이션]        │
│                                                  │
│   HELLO BILLIARD              ┌──────────────┐   │
│   당구 AI 코치                 │  당구대 +    │   │
│                               │  경로 라인   │   │
│   사진으로 읽고,               │  (초록 펠트) │   │
│   물리로 계산한다.             └──────────────┘   │
│                                                  │
└──────────────────────────────────────────────────┘
     ← 좌 55% 텍스트          우 45% 비주얼 →
```

- 로고: `design/logo/Hellobilliard_Logo2.png` 사용
- 비주얼: 3구 경로 계산 화면 캡처를 살짝 기울여 배치(또는 `design/intro.jpg` 활용)
- 글자는 **최소 48px 이상** (썸네일로 축소돼도 읽히게)
- 안전 영역: 가장자리 **각 24px 안쪽**에 중요한 요소 배치

**메인 카피 후보**
- `사진으로 읽고, 물리로 계산한다`
- `당구대 사진 한 장 → 득점 경로`
- `4구 경로 + 3구 시스템, 한 앱에`

## C. 아이콘

- **Play**: 512×512 PNG — `icons/icon-512.png` 그대로 사용 가능 ✓
- **App Store**: 1024×1024 PNG, **알파(투명) 없이** — `icon-512.png`를 1024로 업스케일하거나
  원본에서 재출력. 투명 배경이면 `#100e0a`로 채울 것.

## D. 텍스트 (양 스토어 공용)

**앱 이름:** `Hello Billiard`
**부제(App Store, 30자):** `사진으로 읽는 당구 경로 AI 코치`
**짧은 설명(Play, 80자):**
> 사진 한 장으로 당구 공 배치를 읽고 4구 득점 경로와 3구 시스템을 계산해 주는 AI 코치.

**키워드(App Store, 100자):**
> `당구,쓰리쿠션,3구,4구,당구경로,당구계산,빌리아드,billiard,쿠션,파이브앤하프,당구연습,대회전`

**전체 설명(4000자 이내):**
```
당구대 사진을 찍으면 공 배치를 인식해, 물리 시뮬레이션으로 득점 경로를
성공률과 함께 제안하는 당구 학습 도구입니다.

■ 사진 인식
당구대 사진 한 장이면 공 위치를 자동으로 잡아냅니다.
사진은 기기 안에서만 처리되며 서버로 전송되지 않습니다.

■ 4구 경로 시뮬레이터
실측 분리각·회전(시네루)·다중 경로를 물리 엔진으로 계산합니다.
초급/중급/상급 실력에 맞춰 칠 수 있는 경로만 추천하고,
몬테카를로 시뮬레이션으로 성공률과 공 모임도를 함께 보여줍니다.

■ 3구(쓰리쿠션)
파이브앤하프·플러스·볼 시스템 계산기를 제공합니다.
물리 경로 탐색은 대회전·와리가리(되돌려치기) 같은
정석 코스를 알아보고 우선 추천합니다.

■ 그 밖에
· 설치 없이 가볍게, 오프라인에서도 동작
· 테이블 보정(쿠션·구름)으로 우리 구장에 맞추기
· 한국어 / English
```
